# Docly
Docly is an AI-Powered Documentation Generator that transforms your codebase into professional documentation with a single command. It leverages Google's Gemini AI to analyze your project structure and code, producing comprehensive docs and diagrams automatically.

Why Docly?
Developers often struggle with maintaining documentation. It's time-consuming, repetitive, and often becomes outdated. Docly solves this by automating the entire process.
